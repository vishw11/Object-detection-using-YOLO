# Vehicle Detection in Traffic using YOLO

A deep learning based computer vision project for detecting vehicles in traffic scenes using the YOLO (You Only Look Once) object detection algorithm.

## Overview

This project focuses on automatic vehicle detection in traffic images/videos for intelligent transportation systems. The trained YOLO model identifies multiple vehicle classes and predicts their locations using bounding boxes with confidence scores.

Applications include:

- Smart traffic monitoring  
- Vehicle counting  
- Congestion analysis  
- Autonomous driving support  
- Urban surveillance systems  

---

## Features

- Real-time object detection using YOLO  
- Detects multiple vehicle classes  
- Bounding box localization  
- Confidence score prediction  
- Multi-object detection in a single frame
